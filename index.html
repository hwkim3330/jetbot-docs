Nano vs Orin Nano 비교

왜 Orin Nano로 업그레이드해야 하는가?

사양 Jetson Nano (Maxwell) Jetson Orin Nano (Ampere) 설계 시 함의
GPU 아키텍처 128-core Maxwell 1024-core Ampere 실시간 객체 탐지 + 3D SLAM 동시 수행
AI 성능 0.5 TOPS 40 TOPS (80배 향상) 트랜스포머 모델 구동 가능
CPU 4-core Cortex-A57 6-core Cortex-A78AE ROS 2 멀티스레드 처리 효율 증대
메모리 4GB LPDDR4 4GB / 8GB LPDDR5 고대역폭 센서 데이터 Zero-Copy 처리
운영체제 Ubuntu 18.04 (JetPack 4) Ubuntu 22.04 (JetPack 6) ROS 2 Humble 네이티브 지원
I/O 로직 레벨 3.3V (일부 5V 허용) 3.3V (엄격함) 레벨 시프터 필수!
카메라 커넥터 15-pin (1.0mm) 22-pin (0.5mm) 어댑터 케이블 필수!

Quick Start

빠른 시작을 위한 핵심 단계

1️⃣

부품 준비

• Jetson Orin Nano Developer Kit
• 3S/4S LiPo 배터리 (2200mAh+)
• PCA9685 + TB6612 모터 드라이버
• RPLIDAR A1/A2
• IMX219 카메라 + 22핀 어댑터

2️⃣

OS 설치

• NVMe SSD 준비 (필수 권장)
• SDK Manager로 JetPack 6 플래싱
• CUDA, cuDNN, TensorRT 설치
• 한글 입력 환경 설정

3️⃣

ROS 2 설치

• ROS 2 Humble Desktop 패키지 설치
• rosdep 초기화
• colcon 빌드 도구 설치
• 워크스페이스 생성

4️⃣

하드웨어 연결

• I2C 버스 7번 확인 (Orin Nano)
• 레벨 시프터 배선
• LiDAR udev 규칙 설정
• 카메라 CSI 연결
